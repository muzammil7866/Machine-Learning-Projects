# CART and Random Forest Comparison

## Overview
This notebook compares CART and Random Forest models on both classification and regression tasks using small synthetic datasets. It is designed to show the difference between a single tree and an ensemble-based approach.

## What The Notebook Covers
- CART classification on a toy dataset.
- Random Forest classification on the same data.
- CART regression using DecisionTreeRegressor.
- Random Forest regression using RandomForestRegressor.
- Accuracy, MSE, and R2-based comparison.

## What The Output Shows
- Predictions from the CART classifier and regressor.
- Predictions from the Random Forest models.
- A comparison note stating that Random Forest performed better on the classification example in the notebook.
- Tree diagrams that illustrate the learned split structure.

## Business Value
This kind of comparison helps a team decide when to use a simple, interpretable tree and when to move to a more robust ensemble for better predictive performance.

## Key Takeaways
The notebook shows hands-on understanding of tree-based learning, ensemble methods, and model comparison across both classification and regression.

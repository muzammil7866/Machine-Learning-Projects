# ID3 Decision Tree

## Overview
This notebook implements the ID3 decision-tree idea from scratch using the classic Play Tennis dataset. It manually calculates entropy and information gain instead of relying only on a library implementation.

## What The Notebook Covers
- Importing the required scientific Python libraries.
- Building a categorical training table.
- Converting categories to numeric labels.
- Defining an entropy function with log2.
- Calculating split quality and information gain.
- Extending the reasoning with markdown explanations and tree-related outputs.

## What The Output Shows
- Encoded data tables.
- Entropy and split-computation results.
- Manual reasoning for how ID3 chooses attributes.

## Business Value
ID3-style logic is valuable when a business wants transparent if-then decision rules that can be explained to stakeholders without heavy statistical overhead.

## Key Takeaways
The notebook demonstrates core decision-tree mathematics, feature selection logic, and a strong grasp of explainable machine-learning foundations.

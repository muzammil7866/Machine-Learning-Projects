# Decision Tree Classification

## Overview
This notebook demonstrates decision-tree classification on two separate examples: a small Play Tennis style dataset and a breast-cancer classification workflow built from a CSV file.

## What The Notebook Covers
- Manual feature encoding with LabelEncoder.
- Train and test data preparation.
- DecisionTreeClassifier training and prediction.
- Accuracy evaluation on the held-out data.
- Tree plotting for visual interpretation.

## What The Output Shows
- A simple tree learned from the toy dataset.
- A larger tree learned from the cancer dataset.
- Accuracy printed for both examples.
- Visual tree diagrams that expose the split logic.

## Business Value
Decision trees are useful when interpretability matters. They help teams explain why a record was classified a certain way, which is important in healthcare, operations, policy, and customer decisioning.

## Key Takeaways
The notebook shows explainable modeling, preprocessing, and visual inspection of a tree-based classifier on both toy and real-world style data.
